# 🎨 Google Play Feature Graphic Rehberi

## 📐 Teknik Gereksinimler

- **Boyut:** 1024 x 500 piksel (zorunlu)
- **Format:** PNG veya JPG
- **Dosya Boyutu:** 1 MB'dan küçük olmalı
- **Oran:** 2.048:1 (genişlik:yükseklik)
- **Arka Plan:** Solid color veya gradient (transparent PNG kullanmayın)

## 🎯 Feature Graphic Nedir?

Feature Graphic, Google Play Store'da uygulamanızın sayfasının en üstünde görünen büyük banner görseldir. Kullanıcıların ilk gördüğü görsel olduğu için çok önemlidir.

## 💡 Save All İçin Öneriler

### Tasarım Önerileri:

#### 1. **Ana Tema:**
- **Renkler:** Beyaz arka plan, mavi/vurgu renkleri (uygulama temasına uygun)
- **Stil:** Modern, minimal, profesyonel
- **Odak:** AI teknolojisi ve koleksiyon yönetimi

#### 2. **İçerik Önerileri:**

**Seçenek A: Logo + Slogan Odaklı**
```
[Save All Logo] (sol üst veya merkez)
"Koleksiyonunuzu AI ile Yönetin" (büyük, okunabilir font)
"Fotoğraf Çek • Ürün Tanı • Fiyat Tespit Et" (alt başlık)
[Küçük ikonlar: 📸 🤖 💰]
```

**Seçenek B: Özellik Odaklı**
```
Sol taraf: Save All logosu + "AI Destekli Koleksiyon Yönetimi"
Sağ taraf: 3 küçük görsel/ikon:
  - Fotoğraf çekme ikonu
  - AI analiz ikonu  
  - Koleksiyon görünümü ikonu
```

**Seçenek C: Görsel Odaklı**
```
Arka plan: Gradient (mavi-beyaz)
Merkez: Büyük Save All logosu
Alt kısım: "Smart Collection Management" (İngilizce)
Küçük ikonlar: 📸 🤖 📁 💰
```

### 3. **Renk Paleti Önerileri:**

**Seçenek 1: Mavi-Beyaz (Önerilen)**
- Arka plan: Beyaz veya açık mavi gradient
- Metin: Koyu mavi (#2c3e50) veya siyah
- Vurgu: Mavi (#3498db)

**Seçenek 2: Modern Gradient**
- Arka plan: Mavi'den beyaza gradient (#3498db → #ffffff)
- Metin: Beyaz veya koyu mavi
- Logo: Beyaz veya mavi

**Seçenek 3: Minimal Beyaz**
- Arka plan: Beyaz
- Logo ve metin: Mavi tonları
- İkonlar: Renkli ama minimal

### 4. **Metin Önerileri:**

**Türkçe:**
- Ana başlık: "Save All"
- Alt başlık: "AI Destekli Koleksiyon Yönetimi"
- Veya: "Koleksiyonunuzu Akıllıca Yönetin"

**İngilizce:**
- Main title: "Save All"
- Subtitle: "AI-Powered Collection Management"
- Or: "Manage Your Collection Smartly"

### 5. **Görsel Elementler:**

**İkonlar (küçük, dekoratif):**
- 📸 Kamera (fotoğraf çekme)
- 🤖 Robot (AI teknolojisi)
- 📁 Klasör (koleksiyon yönetimi)
- 💰 Para (fiyat tespiti)
- ⭐ Yıldız (premium özellikler)

**Görsel Stil:**
- Flat design (düz tasarım)
- Minimal ikonlar
- Modern tipografi
- Yeterli boşluk (whitespace)

## 📝 Hızlı Tasarım Adımları (Canva ile)

1. Canva'da "Google Play Feature Graphic" şablonunu seçin
2. Arka planı beyaz veya gradient yapın
3. Save All logosunu ekleyin (sol üst veya merkez)
4. Ana başlığı ekleyin: "Save All"
5. Alt başlığı ekleyin: "AI Destekli Koleksiyon Yönetimi"
6. Küçük ikonlar ekleyin (📸 🤖 💰)
7. Renkleri uygulama temasına uygun yapın
8. Export edin: PNG formatında, 1024x500 piksel
